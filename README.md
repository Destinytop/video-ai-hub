# VideoAI Hub

Multi-Model AI Video Generation Platform Landing Page

## Quick Start

1. Fill in `.env.template` and save as `.env`
2. Deploy to Vercel or any static hosting

## Features

- Multi-model showcase (Seedance, Kling, Veo, Sora)
- Pricing plans
- FAQ section
- Responsive design

## Deployment

### Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

### Manual Deploy

1. Upload `index.html` to any static hosting (Netlify, Cloudflare Pages, etc.)
2. Configure custom domain

## Project Structure

```
ai-video-saas/
├── index.html          # Landing page
├── .env.template      # Configuration template
└── README.md
```
